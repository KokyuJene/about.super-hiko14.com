About the server

Server Overview.

概要

高級ジェネとは @Super Hiko14 がサーバーホストを務めるジェネレーターです。
ジェネレーター機能に加えてランク機能があります。お金を貯めてランクアップ!
色んなジャンルのPvPや建築場もあり、プレイヤーと楽しく交流できます。
by epipi QwQ

History.

歴史

Gen1 — 建国ジェネレーター

Super Hiko14が最初に作ったジェネレーター。yaenTVのジェネレーターで運営になり、その知識を糧に作成。
Owner: Super Hiko14 / OPs: sebinkyo, me_youtube, miochan03

Gen2 — ジェネレーター, PVP, SHOP, 建国 v4.0

Super Hiko14という名前が少し広がったジェネレーター。かなり良い出来だと...思う...。たぶん。
Owner: Super Hiko14 / OPs: yeanTV, kento7587 ほか多数

Gen3 — GEN SHOP PVP

ゴミ。機能不足/改悪化によって誰も来なくなり急遽Gen2に戻された悲しいジェネレーター。
Owner: Super Hiko14

Gen4 — 新\-高級ジェネ-/

黄金期。初めて「高級ジェネ」という名前ができた。初期案は「豪華ジェネ」だったが語呂が悪かったのでkento氏の提案で高級ジェネに。色々神だったが問題も多かった。

Gen5 — 新 高級ジェネ

人気はGen4より少し下がったが機能性が多いジェネレーター。Switch2問題により重くなった苦い思い出。色々なことがあった。
公式サイト → MEMBERS

Gen6 — 新 高級ジェネ™ 3 (現行)

軽く、多機能、楽しいをコンセプトに今もなおアップデートが続く最新バージョン。念密な計画のもと2026.10.1に正式リリース。Discordサーバーや公式サイトも本格化。
公式サイト → MEMBERS
